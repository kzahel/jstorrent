# JSTorrent

A modern BitTorrent client for Chrome, supporting ChromeOS, Mac, Windows, and Linux.

This is an open-source rewrite of the original JSTorrent Chrome App, rebuilt as a Chrome Extension to work with modern browsers after Chrome Apps were deprecated.

## Install

- [Chrome Web Store](https://chromewebstore.google.com/detail/jstorrent/dbokmlpefliilbjldladbimlcfgbolhk)
- [new.jstorrent.com](https://new.jstorrent.com) - Installation page with automatic extension detection

## Features

- Native BitTorrent protocol implementation in JavaScript
- Optional native host for improved disk I/O performance
- Magnet link and .torrent file support
- BitTorrent engine works entirely in the browser
